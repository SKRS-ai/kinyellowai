# Design System

## Theme & Style
- **Mood**: Inclusive, authoritative, modern, and expansive. 
- **Style**: "Hyper-Clean Directory." We are moving away from the 90s "list" feel toward a 2025 "discovery" feel. Using depth through subtle shadows (soft-ui), glassmorphism for search components, and a bento-grid approach for categories.
- **Mode**: Light Mode primary for accessibility and trust, with a "High-Contrast" toggle for inclusivity.

## Color Palette
- **Primary (Yellow)**: #FFD700 (Kin Gold) - Used for high-impact CTAs and branding highlights.
- **Secondary (Dark)**: #0F172A (Slate 900) - Used for deep backgrounds and primary typography.
- **Accent (Culture Blue)**: #3B82F6 - To represent diversity and reliability.
- **Neutral**: #F8FAFC (Slate 50) - Page backgrounds to keep the directory feeling light.

## Typography
- **Headings**: Sora (ExtraBold) - Modern, geometric, and friendly.
- **Body**: Inter (Regular/Medium) - Optimized for high-readability at small sizes for directory listings.
- **Data/Labels**: JetBrains Mono - For state counts and technical data points, adding a modern "indexed" feel.

## Layout & Spacing
- **Grid**: 12-column liquid grid with 24px gutters.
- **Spacing**: Generous whitespace (80px+ between sections) to prevent "directory fatigue."
- **Cards**: Soft radius (16px) with 1px subtle borders (#E2E8F0) and "lift-on-hover" animations.

## Components
- **Buttons**: Pill-shaped with gentle gradients and micro-bounce interactions.
- **Inputs**: Large, 64px height for main search bars with inset shadows and iconography.
- **Badges**: Using semi-transparent backgrounds of the primary color (e.g., Gold/10) for category tags.

## Motion & Animation
- **Philosophy**: Functional motion. 
- **Transitions**: Smooth page fades (300ms) and staggered entrance for grid items. 
- **Hover States**: Cards scale by 1.02x with a blurred glow effect.
